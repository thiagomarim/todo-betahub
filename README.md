## 🚀 Vamos começar

Clone esse projeto e acesse os arquivos.

```bash
git clone https://github.com/thiagomarim/todo-betahub.git

cd todo-betahub

```

Execute esse comando para instalar as dependências.

```bash
npm install

npm start
```
